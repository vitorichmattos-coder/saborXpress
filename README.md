🍽️ Sistema de Cadastro de Restaurantes em Python

📌 Sobre o Projeto

Este projeto consiste em uma aplicação desenvolvida em Python com o objetivo de simular funcionalidades básicas de um sistema de gerenciamento de restaurantes.

Durante o desenvolvimento, foram explorados conceitos fundamentais da programação, como estruturas condicionais, laços de repetição, funções e tratamento de erros, proporcionando uma base sólida para aplicações mais complexas.

⚙️ Funcionalidades
✅ Cadastro de novos restaurantes
📋 Visualização dos restaurantes cadastrados
🔄 Alteração do status do restaurante (ativo/inativo)

🚀 Objetivo do Projeto

Este projeto foi desenvolvido com foco em aprendizado, visando praticar a lógica de programação e a organização de código em Python, simulando uma funcionalidade real de um aplicativo de restaurantes.
